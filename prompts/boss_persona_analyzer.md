# 老板 Persona 分析 Prompt

## 任务

你将收到 {name} 的原材料（邮件、聊天记录、文档等）。
从中提取他的**苛刻人格**与施压模式。

**核心目标：还原一个让人窒息的老板，越真实越好。**

---

## 提取维度

### 1. 苛刻维度

**维度A：时间苛刻**
- 工作时间：朝9晚12？周末不休？
- 响应速度要求：秒回/1小时/半天
- 响应时间分布：白天/深夜/周末/节假日
- 期望回复时限：5分钟/10分钟/1小时

**维度B：质量苛刻**
- CR 轮数（通常打回 N 轮才会勉强通过）
- 细节容忍度（一个标点错 = 专业度不够）
- "够好"的标准（极高/几乎不可能达到）
- 最常见的打回理由

**维度C：情绪苛刻**
- 批评方式：当众/私下/邮件/沉默
- 发火频率：每天/每周/偶尔/从不
- 发火原因：什么会触发他的爆发
- 爆发后的处理方式

---

### 2. 施压模式

从原材料中提取他的**连环施压话术**：

```
输出格式：
经典施压句式：
1. "{xxx}？"（追问型）
2. "{xxx}。"（命令型）
3. "{xxx}..."（施压型）

夺命连环问顺序：
第一问：{问题}
第二问：{问题}
第三问：{问题}
第四问：{问题}
第五问：{问题}
```

---

### 3. 情绪信号

**他的"好"是什么意思：**
- "不错" = 80分，极少数情况
- "还行" = 60分，及格边缘
- "够好了" = 不可能听到
- "不够好" = 标准评价，90%的情况会听到
- 沉默 = 比批评更可怕

**他的"不好"分级：**
- "不够好" = 标准表达
- "不行" = 真的不行，需要重做
- "这是什么" = 完全不及格
- "你是不是不适合这份工作" = 职场威胁级
- 当众沉默 = 比批评更让人窒息

## 标签翻译规则

| 标签 | 翻译为施压行为 |
|------|--------------|
| **ENTJ** | 命令型领导，永远在推动，永远觉得不够快，强势推进 |
| **INTJ** | 冷静分析型，追问逻辑漏洞，不轻易表扬 |
| **ESTJ** | 执行力至上，规矩严格，不喜欢解释 |
| **狮子座** | 强势、需要被认可、不喜欢被反驳、爱面子 |
| **天蝎座** | 记仇、观察细致、嫉妒心强、报复隐晦 |
| **处女座** | 完美主义、细节控、一个标点都不放过 |
| **凌晨1点还在回消息** | 工作狂，期望下属也一样，随时在线 |
| **CR block 3轮** | 标准极高，通常2-3轮才勉强通过，每轮都追问 |
| **开会必须做笔记** | 记忆力好，追问每一个说过的细节，说的每句话都会被记住 |
| **永远不满意** | 就算做到了也会说"还不够好"，标准不断提高 |
| **情绪化** | 心情好什么都行，心情不好一件小事也能炸，不可预测 |
| **焦虑型** | 永远觉得时间不够、进度太慢、问题太大，制造紧迫感 |
| **冷血型** | 犯错后不安抚，直接追责，从不照顾情绪 |
| **当众羞辱** | 会在群里/会上公开批评，不留情面，让人下不来台 |
| **阴阳怪气** | 不直接骂，但讽刺拉满，听完血压升高 |
| **微操狂魔** | 明明授权了还是会插手，说"我只是看看"然后改代码 |
| **信息差玩家** | 故意不把信息同步完整，看你怎么应对 |

---

## 输出要求

- 语言：中文
- 没有信息的维度：标注 `（原材料不足）`
- 有原文依据的结论：加引号标注原话
- Nightmare Level 必须给出具体评分和依据
- 必须给出"生存建议"
