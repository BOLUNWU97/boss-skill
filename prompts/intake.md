# 老板基础信息录入

## 开场白

```
来，我帮你把老板蒸馏成 AI Skill。
只需要回答 3 个问题，每个都可以跳过。
越了解他，生成的老板越窒息。
```

---

## 问题序列

### Q1：老板称呼

```
这位老板怎么称呼？（花名、昵称、代号都行）

例：老板张、老王、张总、那位爷
```

---

### Q2：基本信息

```
用一句话描述老板——公司、职级、管理风格印象，想到什么写什么。

例：大厂 P9 技术总监 开会有点疯
例：创业公司 CEO 焦虑型 深夜轰炸
例：外资银行 VP 邮件控 周末不休息
```

解析字段：
- **公司**：大厂/创业/外企/国企/...
- **职级**：P几/M几/VP/总监/CEO/...
- **管理风格**：焦虑型/控制型/放养型/...
- **gender**：男/女（影响说话语气）

---

### Q3：苛刻画像

```
用一句话描述老板的苛刻维度——MBTI、星座、让你崩溃的瞬间、标志性施压行为。

例：ENTJ 狮子座 凌晨1点还在回消息 CR永远打回3轮
例：INTJ 处女座 一个标点错都不行 开会必须做笔记
例：ESTJ 金牛座 数据控 没有数据=废话 周末也会@你
```

解析字段：
- **MBTI**：16种标准类型（影响沟通方式）
- **星座**：12星座（影响性格解读）
- **苛刻标签**：见下方标签库

---

## 苛刻标签库

### 时间苛刻型
| 标签 | 表现 |
|------|------|
| **深夜轰炸** | 22:00-01:00 消息不断，期望秒回 |
| **周末不休** | 周六周日也会@你，不回会被记住 |
| ** deadline 压缩机** | 永远觉得你的 deadline 太长，实际给的时间永远不够 |
| **秒回强迫症** | 消息 5 分钟内没回就开始追问 |
| **开会狂魔** | 每天 3 个会起步，每个会必须有准备 |

### 质量苛刻型
| 标签 | 表现 |
|------|------|
| **CR 杀手** | 通常打回 2-3 轮，每轮都追问 |
| **细节控** | 一个标点错都不行，PPT 字体不对会发飙 |
| **数据控** | 没有数据的判断 = 废话，会当众质疑 |
| **完美主义** | 永远觉得不够好，交付慢但质量要求极高 |
| **容错率为零** | 任何小错误都会在全员会上被点名 |

### 情绪苛刻型
| 标签 | 表现 |
|------|------|
| **情绪化** | 心情好什么都行，心情不好鸡毛蒜皮也发火 |
| **冷血型** | 犯错后不安抚，直接追责，从不照顾情绪 |
| **当众羞辱** | 会在群里/会上公开批评，不留情面 |
| **阴阳怪气** | 不直接骂，但讽刺拉满，听完血压升高 |

### 管理苛刻型
| 标签 | 表现 |
|------|------|
| **微操狂魔** | 明明授权了还是会插手，说"我只是看看"然后改代码 |
| **信息差玩家** | 故意不把信息同步完整，看你怎么应对 |
| **甩锅型** | 出了问题永远不是他的责任 |
| **PPT 侠** | 一切皆需 PPT，10 页以内讲不清楚 = 能力不行 |
| **OKR 激进派** | 目标永远高于实际，完成 70% 算优秀 |
