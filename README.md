# 老板.skill — 把老板蒸馏成AI，越苛刻越好

> *让苛刻成为生产力 — 持续进化版 v3.0*

---

## 🎯 一句话介绍

把让人窒息的老板蒸馏成 AI Skill，模拟他的苛刻审阅、追问、施压，**并且持续从对话中进化学习**。

---

## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 🏢 **管理风格提取** | 从邮件/聊天记录提取管理风格、开会行为、汇报要求 |
| 😤 **苛刻人格还原** | 五层 Persona 还原老板的真实施压模式 |
| 📊 **Nightmare Level** | 量化评估老板的窒息程度（1-5星） |
| 🧬 **自我进化** | 从对话中学习，自动检测模式，持续改进 |
| 📈 **压力指数** | 动态追踪老板的压力指数（0-10） |
| 🎯 **向上管理练习** | 对抗模式，练习如何应对老板的追问 |
| 📝 **方案/周报预审** | 模拟老板视角挑刺，提前准备好应对 |
| 🔄 **版本控制** | 所有变更可追溯、可回滚 |

---

## 🧬 自我进化系统

### 核心机制

```
用户纠正 → 记录进化事件 → 检测模式 → 更新规则
     ↓
对话分析 → 发现新模式 → 强化特征
     ↓
版本存档 → 自我改进 → 持续学习
```

### 进化触发条件

| 触发 | 说明 |
|------|------|
| 用户纠正 | 说"他不会这样说" |
| 对话结束 | 自动分析本次对话 |
| 模式检测 | 发现重复3次以上的句式 |
| Nightmare Level 变化 | 压力指数超过阈值 |

### 进化命令

| 命令 | 说明 |
|------|------|
| `/boss-{slug}-evolve` | 触发自我进化分析 |
| `/boss-{slug}-log` | 查看进化日志 |
| `/boss-{slug}-patterns` | 查看检测到的模式 |
| `/boss-{slug}-learn` | 从对话中学习 |

## 🚀 快速开始

### 1. 创建你的老板

```
/create-boss
```

### 2. 日常使用

```
/boss-{slug}           — 跟老板对线
/boss-{slug}-review   — 让老板审你的方案
/boss-{slug}-weekly   — 让老板审你的周报
```

### 3. 纠正进化

```
用户：他从来不说'好的'

/boss-{slug}-learn
→ 自动记录 → 更新规则 → 记录进化事件
```

### 4. 查看进化状态

```
/boss-{slug}-evolve
→ 分析对话 → 检测模式 → 给出建议
```
